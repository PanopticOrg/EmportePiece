# Emporte Piece: Extract Images from Photo Albums

Simple tool using FastAPI for the backend and React for the FrontEnd. Upload images, see the extractions proposed by the YOLO model, correct them if need and it will then extract the images in the "output" folder and create training files to finetune the YOLO if needed.

## Install

- Install the requirements.txt (ideally in an env)
- then run `python main.py`
  
The app should then be available at http://localhost:8001
